# bank-indicator

`bank-indicator` is the internal web component used by `jb-payment-input` to display detected bank information for supported card or account values.

## Usage
`bank-indicator` is usually rendered by `jb-payment-input`. Use it directly only when you need the bank indicator UI without the full payment input.

## CSS Variables
| CSS variable name | description |
| --- | --- |
| --bank-indicator-padding | Bank logo padding. |
